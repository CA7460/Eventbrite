Core features like Login/auth, walkthrough screens (Screens which are only visible at after the install only), 
application setting features are the core features that should be added in a folder Core.

1-  auth: Auth folder must contain the features: Register, Login, Forgot password.

2-  walk_through: Walkthrough screens must contain all the screens which will be visible 
    only when the application starts for the first time after the fresh install.

3-  settings: This will be the application setting feature
